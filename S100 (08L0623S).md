### CLP controlando ambos os inversores via Ethernet TCP Modbus(O mais comum)

Este método é comum onde o CLP Envia Comandos de: 

*Velocidade;*
*Start/Stop;*
*Reversão;*
*Aceleração;*
*Desaceleração;* 

E envia comandos de Leitura: 

*Corrente;* 
*Frequência;* 
*Torque;*
*Alarmes.* 


Este método é constantemente utilizado em Esteiras, máquinas de Embalagens, Transportadores e Linhas de montagem. 

**Atividade prática: [[Controle de Dois inversores de Frequência Via TCP Modbus]]**

A ideia é comunicar Via [[Ethernet TCP Modbus]]. Estarei utilizando um CLP [[XBC-DN32UP (15L3998S)]] Este que já tem entradas para cabo Ethernet(Rj45) de forma nativa facilitando para a conexão dos dois inversores de frequência.
